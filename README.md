# CHANEL Fragrances

## Descripción General

CHANEL Fragrances es un sitio web de comercio electrónico desarrollado como proyecto académico. La aplicación simula una tienda online especializada en perfumes de la marca CHANEL, permitiendo a los usuarios explorar una colección de fragancias, agregarlas a un carrito de compras y completar un proceso de compra mediante un formulario de datos del cliente.

El proyecto fue diseñado con una interfaz moderna, elegante y responsiva, inspirada en el estilo visual de las marcas de lujo.

---

## Objetivo del Proyecto

El objetivo es simular una experiencia básica de compra en línea mediante una aplicación web desarrollada con tecnologías frontend, implementando funcionalidades dinámicas utilizando JavaScript y almacenamiento local del navegador.

## Funcionalidades Implementadas

### Catálogo de Productos

* Visualización de perfumes CHANEL.
* Información básica de cada producto.
* Diseño de tarjetas responsivo.

### Carrito de Compras

* Agregar productos al carrito.
* Incrementar cantidades.
* Disminuir cantidades.
* Eliminar productos individuales.
* Vaciar carrito completo.
* Cálculo automático del total.

### Persistencia de Datos

* Uso de LocalStorage para conservar los productos agregados incluso al recargar la página.

### Formulario de Compra

* Registro de datos del cliente.
* Validación de campos obligatorios.
* Selección de método de pago.
* Confirmación de compra.

### Diseño Responsivo

* Adaptación para dispositivos móviles.
* Adaptación para tabletas.
* Adaptación para computadoras de escritorio.
